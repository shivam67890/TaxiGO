About Taxigo

Welcome to Taxigo, a college project aimed at revolutionizing urban transportation through a user-friendly cab booking platform. Our goal is to create an efficient and reliable system that connects passengers with drivers seamlessly.

This project was developed as part of our final year computer science curriculum, focusing on modern web technologies and real-world application development. We've implemented features like real-time booking, GPS tracking, and secure payment integration to create a comprehensive solution.
